Steps on how to ro run this Ground Water Analysis system server:

* Open folder on visual studio code
* Install all dependencies inside the requirements.txt file by running "pip install -r requirements.txt" in your terminal.
* After successfully installing the dependencies,on your terminal, locate the app.py file by typing the following prompt on the terminal
     "cd backend"

* Run the backend server by typing "python app.py" 

NOTE:

This system is currently running on a development server not production as yet.
